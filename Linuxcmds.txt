Navigating the File System:

ls: List files and directories in the current directory.
cd: Change directory.
pwd: Print the current working directory.
mkdir: Create a new directory.
rmdir: Remove a directory.
cp: Copy files and directories.
mv: Move or rename files and directories.
rm: Remove files or directories.


File Management:

cat: Concatenate and display file content.
head: Display the beginning of a file.
tail: Display the end of a file.
less or more: View file content page by page.
touch: Create an empty file or update file timestamps.
nano or vim: Text editors for creating or editing files.


File Permissions:

chmod: Change file permissions.
chown: Change file ownership.
chgrp: Change file group ownership.
sudo: Execute commands with superuser (root) privileges.


Package Management:

apt: Package manager for Debian-based systems like Ubuntu.
apt-get: Command-line tool for managing packages.
dpkg: Low-level package manager for Debian-based systems.


Process Management:

ps: Display information about active processes.
top or htop: Interactive process viewer.
kill: Terminate a process.
killall: Terminate processes by name.
pgrep: Search for processes by name.
System Information:
uname: Display system information.
hostname: Display or set the system's hostname.
df: Display disk space usage.
free: Display memory usage.
uptime: Display system uptime.
